# Calculadora Web

Una calculadora funcional desarrollada con tecnologías web estándar. Este proyecto permite realizar operaciones matemáticas básicas mediante una interfaz minimalista y eficiente.

## Características

* **Operaciones matemáticas:** Soporta suma, resta, multiplicación y división.
* **Controles de edición:** Incluye un botón "AC" para reiniciar la pantalla y "DEL" para eliminar el último carácter ingresado.
* **Lógica de cálculo:** Evalúa expresiones matemáticas y permite continuar operando sobre el resultado obtenido.
* **Interfaz adaptable:** El diseño utiliza CSS Grid para mantener una cuadrícula organizada de botones.
* **Gestión de errores:** Muestra un mensaje de "ERROR" en caso de introducir operaciones no válidas.

## Tecnologías

* **HTML5:** Define la estructura de la calculadora y sus componentes.
* **CSS3:** Gestiona el diseño visual, los colores y la disposición de los elementos en pantalla.
* **JavaScript:** Implementa la lógica de procesamiento de datos y la interactividad de los botones.

## Instrucciones de Uso

Para ejecutar el proyecto localmente:

1. Descarga o clona los archivos del repositorio.
2. Abre el archivo `index.html` en tu navegador preferido.
2.5 O en otro caso clonalo en tu propio IDE con el siguiente comando.<br>
`git clone https://github.com/xclipsy/MyFirstCalculator.git`

## Estructura de Archivos

* `index.html`: Estructura del documento.
* `styles.css`: Hoja de estilos y diseño de la cuadrícula.
* `script.js`: Lógica y funcionamiento interno.
